Mode d'emploi

Bienvenue sur le Timer de Billard Pro v4 ! Cette application est conçue pour arbitrer vos parties avec précision et simplicité, en respectant les règles officielles du billard international 8-ball.

📱 Interface Principale

  • Chronomètre Central : Affiche le temps restant pour le coup en cours. Il change de couleur selon l'urgence :
    • ⚪ Blanc : Temps normal
    • 🟠 Orange : Alerte (paramétrable, défaut 15s)
    • 🔴 Rouge : Critique (paramétrable, défaut 5s)
  • Cartes Joueurs (P1/P2) : En haut de l'écran
    • Affiche le joueur actif (surligné avec sa couleur)
    • Indique le statut de l'extension : EXT (disponible) ou EXT (utilisée)
    • Cliquez sur une carte pour changer de joueur
  • Barre Extension : Ajoute du temps supplémentaire (1x par joueur par partie)

🎮 Contrôles & Actions

Écran Tactile (Zone du chrono)

  • Simple Clic : Lance ▶️ ou met en pause ⏸️ le chronomètre
  • Double Clic : Réinitialise le temps pour le joueur actuel

Cartes des Joueurs

  • Clic sur P1 ou P2 : Change de joueur et réinitialise automatiquement le timer
  • Le joueur actif est mis en évidence avec sa couleur personnalisée

Panneau de Contrôle (boutons du bas)

  • 🔄 Reset : Réinitialise le timer pour le joueur actuel
  • ▶️/⏸️ Play/Pause : Démarre ou met en pause le chronomètre
  • NEW : Nouvelle partie (réinitialise tout, y compris les extensions)

Extension de Temps

  • Chaque joueur peut utiliser 1 extension par partie
  • Cliquez sur la barre EXTENSION pendant que le timer tourne
  • Ajoute immédiatement le temps d'extension configuré (défaut : 15 secondes)
  • La barre devient rouge une fois utilisée pour le joueur

⌨️ Raccourcis Clavier

Contrôlez le timer rapidement avec votre clavier :

  • Espace : Démarrer / Mettre en pause
  • R : Réinitialiser le coup (Reset)
  • N : Nouvelle partie
  • 1 / ← : Sélectionner Joueur 1
  • 2 / → : Sélectionner Joueur 2
  • E / ↑ : Utiliser l'extension
  • M / Echap : Ouvrir / Fermer le menu
  • F : Activer / Désactiver le plein écran

⚙️ Configuration (Roue crantée)

Accédez aux paramètres en cliquant sur l'icône ⚙️ en haut à droite :

  • Plein Écran : Le bouton pour basculer en plein écran se trouve dans l'en-tête de ce panneau.
  • Joueurs : Personnalisez les noms (max 5 caractères) et couleurs
  • Temps :
    • Temps de base : 10-120 secondes (défaut 45s)
    • Temps d'extension : 5-60 secondes (défaut 15s)
  • Alertes : Définissez les seuils pour les alertes visuelles orange et rouge
  • Interface :
    • Mode Boutons : Affiche les contrôles en bas
    • Mode Tactile : Cache les boutons pour plus d'immersion
  • Audio : Ajustez le volume des alertes sonores
  • Automatisation : Activez le redémarrage automatique du timer après un reset

📋 Règles Officielles (International 8-Ball)

  • Temps par coup : 45 secondes (configurable)
  • Extension : 1 par joueur par partie
  • Changement de joueur : Après faute, empochage de la blanche, ou échec d'empochage
  • Le timer commence quand toutes les billes sont immobilisées

💡 Conseils d'utilisation

  • En mode tactile, l'écran entier devient interactif
  • Les vibrations nécessitent l'autorisation du navigateur
  • Vos paramètres sont sauvegardés automatiquement
  • Le double-clic reset est idéal pour les corrections rapides
  • Utilisez les cartes joueurs pour un changement rapide et visuel

🎱 Bonne partie !

Procédures d'appel

Approuvé et reconnu comme guide officiel et procédures d’appel par la WORLD EIGHTBALL POOL FEDERATION et ULTIMATE POOL.

L’intention de ce document est d’offrir aux arbitres des conseils et une structure claire sur la façon d’arbitrer un match. Les joueurs peuvent également le lire afin de se familiariser avec les procédures d’appel d’un arbitre.

Vous êtes responsable de faire respecter :

  • Le code vestimentaire requis.
  • L'état d'ébriété des joueurs.
  • Les lois/règles qui régissent le jeu de billard.

Notes sur les appels:

  • Votre appel est en gras.
  • Les mots en italique sont des indications.

APPEL : Comptez les joueurs : 3, 2, 1, tirage au sort.

Après avoir décidé du vainqueur, APPEL : (nom du joueur) gagne le tirage au sort.

APPEL : 1ère manche, (nom du joueur) à la casse.

APPEL : Manche (numéro), (nom) à la casse, score (X) à (Y).

Aucun appel requis.

APPEL : Temps mort, replacez la bille 8, temps en cours.

APPEL : Casse fautive, ré-rack. Le joueur entrant choisit de casser.

APPEL : Faute, bille en main... (depuis la zone si à la casse, sinon de n'importe où).

Si le choix n'est pas évident et que le joueur ne nomine pas, APPEL : Perte de tour.

APPEL : (Jaunes/Rouges) en jeu. (uniquement à la première occasion)

APPEL : Faute, bille en main.

Si le coup est manqué, APPEL : Perte de tour.

APPEL : Faute, perte de manche.

À 5 secondes restantes, APPEL : 5, 4, 3, 2, 1. Si dépassé, APPEL : Faute de temps.

Si demandée, APPEL : Extension demandée. Si non disponible, APPEL : Aucune extension restante.

APPEL : Temps mort. Après résolution, APPEL : Temps en cours.

APPEL : Bille touchée (indiquer laquelle).

APPEL : Bille bloquée (indiquer laquelle).

Si demandée, évaluer et APPELER : Impasse ou Pas d'impasse.

APPEL : Manche.

APPEL : Manche et match (Nom du joueur). Si le temps est écoulé, le plus haut score gagne. Si égalité, possibilité de shoot-out.
